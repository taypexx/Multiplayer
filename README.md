## About

Muse Dash Multiplayer (MDMP) is a project that lets you play Muse Dash together with your friends. It has its own mechanics and features.
We have the profile system, custom achievements, lobbies, queueing (not yet) and much more!
If you wish to help the project contact @cvle. or @taypexx on discord.

## Progress

Right now everything is still under development.
Although the mod is not public yet, sometimes we host testing sessions on our [discord server](https://mdmp.online/discord).
We also share our progress and make polls here.

## Download

1. Install MelonLoader v0.6.1.
2. Make sure you have the latest [Custom Albums](https://github.com/MDMods/CustomAlbums/releases) and Headquarters installed.
3. Get the [libs](https://mdmp.online/libs) needed for Multiplayer and extract them into your mods folder. You don't need to update them in the future.
4. Get the latest Multiplayer version [here](https://github.com/taypexx/Multiplayer/releases).

Mod also supports [FavGirl](https://github.com/ALLMarvelous/FavGirl).
